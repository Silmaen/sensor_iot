Add a new component to the hardware inventory in `docs/components.md`.

Ask me for:
1. Component name / reference
2. Category (MCU, sensor, shield, 18650 cell, power/protection, passive, connector, etc.)
3. Key specifications (voltage, capacity, interface, value, etc.)
4. Quantity available (if known)
5. Any notes (source, color, markings, measured values, etc.)

Then:
- Add the component to the correct section in `docs/components.md`, creating a new section if the category doesn't exist yet
- For 18650 cells, add to the "18650 Cell References" sub-table under Power
- Keep all table columns perfectly aligned (project convention: full-width separators, no spaces around dashes)
